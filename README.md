### Mark workstation blocks and beds to stop villagers from using your things!

## This mod features:
- **Shift clicking a villager workstation block using a *stick* (or configurable to be any `minecraft: item_ID`) ‎‎‎to "*mark*" it, blocking villagers from using them.**
- ***Workstation Marking* and *Bed Marking* config toggles, `enabled` by default**
- ***Pathfinding Tweaks* to stop villagers from getting stuck on trapdoors, `disabled` by default**
- ***Open Config Screen*, *Show Markings*, and *Require Sneaking* config toggle keybinds**
- **Functions on *both* client and server side, *optional* on client but required to edit the config**
